## 简介

OAG是一家航空大数据信息公司。OAG成立于1853年，于1929年开始专注于全球航班数据，总部位于英国，并在全球多处设置办公机构，服务全球客户。  

OAG公司管理及维护全球航班数据，管理并发布全球民航客运和货运航班计划和航班动态数据，被行业视为全球最完整最及时精确的航空信息提供商。  

数据覆盖全球近1000家航空公司和4000多个机场，为全球政府机构、航空公司、机场、高等院校、科研机构、物流企业、OTA企业、咨询机构等众多客户提供航空数据和解决方案支持。OAG以数据全面性、准确性、及时性成就在行业的领导地位。  

## 数据特征

User Guide、Reference

### OAG Schedules Analyser（航班运力数据）

查看全球市场的历史和未来航班运力/计划数据。

- 长度：27年 1996-2024 （长度固定，若在2024.1.1后，1996年的数据就不支持查询了）  

- 最小统计单位：天  

- 未来数据每周更新

- 提供未来一年的运力趋势预测，从2019.2.4 开始 Snapshot（不覆盖原来的未来预测）

### OAG Traffic Analyser （基于订票数据的旅客流量数据）

基于全球GDS的MIDT航空旅客订票数据，用于分析航线和整个航空网络月度旅客流量，能够了解旅客流量的起点、中转点和目的地。

- 长度：13年 2010-2024（长度固定，若在2024.1.1后，2010年的数据就不支持查询了）  

- 最小统计单位：月

- 未来数据每月更新

- 显示未来一年的订票趋势预测，从2021.10 开始Snapshot（不覆盖原来的未来预测）

- 基于航班订座数据（MIDT）

- 包含各舱位收益和价格信息数据

- 按航线评估单向或双向流量

- 可以查看每段航路的月平均票价

## Schedules Analyser功能模块


### 航班运力概览模块 / Capacity Report

提供不同维度的设置，如航班时刻、航空公司、航空联盟、出发/到达机场、城市、国家、地区、机型和时间段（日期、周、月、季节或年份）等用以查看航班运力。

### 航班波模块 / Bank Structure Report

给定时间段内机场航班波情况分析。

### 运力同比环比模块 / Schedule Changes Report

比较两个时间段的航班运力，包括新航线、停止航线、运力变化、机型变化、出发/到达时间时间变化等。

### 最佳排名模块 / Top X Report

根据各种指标（如ASK、ASM、ATK、ATM、座位数、班次数）排名航空公司、出发/到达机场、航线或机型等。

### 客户自定义模块 / Power Table

能够根据所选择的时间和各种指标来创建完全定制的客户化报告。


## Traffic Analyser 功能模块

### O&D流量分析模块 / O&D Report

O & D ： 直航

根据起点和目的地航线的机票预订和旅客流量分析，最多2个中转点。

### Segment航段流量衔接分析模块和MIX全航线分析模块 / Segment Report & Mix Report

Segment ：中转流量

通过Local、Behind、Beyond和Bridge4种不同设置，客户任意选择通航的两点，系统自动匹配四种不同设置下的机票预订和旅客流量数据。

### 旅客中转衔接分析模块 / Connections Report

旅客流量流向中的中转衔接情况。

### 最佳排名模块 / Top X Report

根据各种指标（如订票、舱位、总收入、平均票价）排名航空公司、出发/到达机场、航线、枢纽机场等。

### 客户自定义分析模块 / Power Table
客户根据OAG设定的全球用户最常用的一部分筛选条件自定义查询订座和各舱位票价信息，进行流量分析。
